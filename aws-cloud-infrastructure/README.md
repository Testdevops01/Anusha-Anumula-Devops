# AWS Cloud Infrastructure (Terraform)

## Overview
This project demonstrates infrastructure provisioning on AWS using
Terraform as Infrastructure as Code.

---

## Resources Created
- EC2 Instance
- Security Group
- IAM Role (optional)

---

## Technologies
- AWS EC2
- IAM
- Terraform
- Linux

---

## Workflow
1. Define infrastructure using Terraform
2. Initialize Terraform
3. Plan and apply infrastructure
4. Deploy applications on provisioned resources

---

## Note
Infrastructure code is simplified for learning and portfolio purposes.
Sensitive values are excluded.
